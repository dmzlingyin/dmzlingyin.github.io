如何将Hugo托管到github

December 06, 2022
  1. 登录GitHub并且创建一个特殊仓库: name.github.io(name为你的用户名)
  2. 使用hugo创建一个新的项目, 并删除项目中的public目录
  3. 进入themes目录, 选中你想要的主题: git clone xxxx.git
  4. 修改根目录下的confit.toml文件, 指定主题名称
  5. hugo server 测试是否显示正常
  6. 在项目根目录, git clone name.github.io public
  7. 执行hugo命令, 之后所有的静态文件会被安装到public目录下
  8. 进入public, git push到GitHub
  9. 等待两分钟, 即可访问name.github.io
